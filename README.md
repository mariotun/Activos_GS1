# Activos de GS1


## Detalles Tecnicos

- Comando para iniciar un proyecto en Nodejs
  ```
  npm init --y
  ```
    
- Instalacion de dependencias en el proyecto de Nodejs
  ```
  npm install express mysql body-parser cors
  ```
